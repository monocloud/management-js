    Module @monocloud/api-management

    Introduction

    MonoCloud Management API SDK for Javascript ‚Äì programmatically manage apps, policies, configurations via the MonoCloud APIs.

    MonoCloud is a modern, developer-friendly Identity & Access Management platform.

    This SDK provides a full-featured, typed JavaScript client for interacting with the MonoCloud Management APIs, allowing you to automate tenant administration programmatically.

    üìò Documentation

    • Documentation: https://www.monocloud.com/docs
    • SDK Docs: https://www.monocloud.com/docs/apis/management
    • API Reference: https://monocloud.github.io/api-sdk-js

    Supported Platforms

    • Node.js >= 11.0.0

    üöÄ Getting Started

    Requirements

    • A MonoCloud tenant
    • A Tenant API key with appropriate permissions

    üì¶ Installation

    npm install @monocloud/api-management

    Usage

    const managementClient = MonoCloudManagementClient.init({
      domain: 'https://<your-tenant-domain>',
      apiKey: '<your-api-key>',
    });

    ‚ö†Ô∏è Security Note: Do not hardcode your API key. It is recommended to load it from an environment variable. See: monocloud-management-client

    ‚ú® Usage Examples

    The SDK closely mirrors the REST API structure ‚Äî clients are organized by resource areas (clients, resources, etc.).

    üîç Get all clients

    const result = await managementClient.clients.getAllClients(
        page: 1,
        size: 10,
        filter: "dashboard",
        sort: "name:1"
    );

    Explore further operations at https://www.monocloud.com/docs
