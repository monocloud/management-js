    Module @monocloud/management-admin

    Introduction

    MonoCloud Management Admin SDK for Javascript ‚Äì programmatically manage apps, policies, configurations via the MonoCloud Management Admin APIs.

    MonoCloud is a modern, developer-friendly Identity & Access Management platform.

    This SDK provides a full-featured, typed JavaScript client for interacting with the MonoCloud Management APIs, allowing you to automate tenant administration programmatically.

    üìò Documentation

    • Documentation: https://www.monocloud.com/docs
    • SDK Docs: https://www.monocloud.com/docs/apis/management
    • API Reference: https://monocloud.github.io/management-js

    Supported Platforms

    • Node.js >= 11.0.0

    üöÄ Getting Started

    Requirements

    • A MonoCloud tenant
    • A Management API key with appropriate permissions

    üì¶ Installation

    npm install @monocloud/management-admin

    Usage

    const adminClient = MonoCloudAdminClient.init({
      domain: 'https://<your-tenant-domain>',
      apiKey: '<your-api-key>',
    });

    ‚ö†Ô∏è Security Note: Do not hardcode your API key. It is recommended to load it from an environment variable. See: monocloud-admin-client

    ‚ú® Usage Examples

    The SDK closely mirrors the REST API structure ‚Äî clients are organized by resource areas (clients, resources, etc.).

    üîç Get all clients

    const result = await adminClient.clients.getAllClients(
        page: 1,
        size: 10,
        filter: "dashboard",
        sort: "name:1"
    );

    Explore further operations (clients, options, trust stores, etc.) using the same patterns. See: https://www.monocloud.com/docs

    Explore further operations at https://www.monocloud.com/docs
